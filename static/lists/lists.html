Lists

A list is an example of an ADT. It is abstract as it has high levels of functionality and optimisation. However, this is hidden from the user.

A static size means the size will never change. If a lists size, or data, will never change then it should be an array. Most programming languages have functionality for both arrays and lists.

Examples of lists could include:

  • Students in a class
  • Song lists
  • Available products in a shop
