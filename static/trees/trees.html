Trees

A tree is an ADT with a root, branches and leaves. A tree is a connected, undirected graph with no cycles.

A binary tree is a tree were each node has at most two children. A common use case for a binary tree is for Huffman Encoding. Where each character is represented by a string of binary.

There are three main ways to traverse a binary tree. Pre-order, in-order and post-order.

  • Pre-order begins at the root, then traverses the left sub-tree, then the right sub-tree.
  • In-order begins at the left sub-tree, then the root, and then the right sub-tree.
  • Post-order begins at the left sub-tree, then the right sub-tree and then the root.
A binary tree may be represented as an array of records. Each storing its own data, as well as the other nodes connected to it.
